# MachineVision_CourseWorks

Coursework collection for a Machine Vision / Computer Vision class, implemented in Jupyter notebooks.

> **Academic project archive.**  
> This repository contains assignment work completed for learning purposes. It is kept public as a record of progress across core vision topics.

## Repository structure

- `A1-Imaging` — image fundamentals and basic processing
- `A2-Color` — color spaces and color-based analysis
- `A3-Features` — feature detection / description
- `A4-Motion` — motion analysis and tracking concepts
- `A5-Hough` — Hough-based line/shape detection
- `A6-Triangulation` — geometry and triangulation
- `A7-Recognition` — classical recognition pipelines
- `A8-Deep` — deep learning for vision tasks


## How to run

1. Clone the repository.
2. Open the assignment folder of interest (`A1` … `A8`).
3. Launch notebooks in Jupyter Lab/Notebook.
4. Run cells in order from top to bottom.
